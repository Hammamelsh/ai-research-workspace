# AI Research Workspace

> Upload documents, ask AI questions, generate reports, and perform web research with citations.

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js (React) |
| Backend | Python FastAPI |
| AI / RAG | Amazon Bedrock + LangChain |
| Vector Store | PostgreSQL + pgvector |
| Storage | AWS S3 |
| Infrastructure | Terraform |
| DevOps | Docker, GitHub Actions |

## Environments

| Branch | Environment | Purpose |
|---|---|---|
| `main` | Production | Live, stable releases only |
| `staging` | Staging | Pre-release QA and validation |
| `dev` | Development | Integration of all features |
